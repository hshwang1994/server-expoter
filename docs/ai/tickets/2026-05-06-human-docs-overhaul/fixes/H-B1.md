## 사용자 의도

본 cycle 사용자 명시: 사람이 처음 읽었을 때 막힘 없이 따라갈 수 있도록.

## 작업 범위

| 항목 | 내용 |
|---|---|
| 영향 파일 | `docs/01_jenkins-setup.md` |
| 영향 독자 | 운영자 / 호출자 |
| 핵심 | 처음부터 끝까지 따라가는 step-by-step |

## 변경 항목

1. 도입부 강화 — 이 문서 한 번 읽으면 무엇을 할 수 있는지 명시
2. 사전 준비 박스 (필요한 권한 / 도구 / IP 정보) 보강
3. 명령어 블록마다 "이 명령은 무엇을 하는가" 한 줄 코멘트
4. 잘 막히는 지점 → "트러블슈팅" 절로 분리
5. 약어 풀이 / 내부 cycle 번호 본문 0건
6. 다음 단계 안내 (이 문서 다음에 무엇을 읽어야 하는지)
