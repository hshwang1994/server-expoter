# F14 — Dell iDRAC 10 (17G PowerEdge, 2024-) 호환성

**우선**: P3 / **영역**: system / **출처**: R2 / **원칙**: Additive (신규 adapter)

## 컨텍스트
Dell 17G PowerEdge (2024-) 가 iDRAC 10 적재. firmware 8.x+. 현재 adapter `dell_idrac9.yml` firmware_patterns `^[4-7]\\.` 매칭 안 됨.

## 현재 동작
iDRAC 10 host → adapter_loader가 `dell_idrac.yml` (priority 10) 또는 generic fallback. 본 수집 대부분 통과 (DMTF 표준 호환).

## 변경 (Additive)
새 adapter `adapters/redfish/dell_idrac10.yml` 작성:
```yaml
adapter_id: redfish_dell_idrac10
priority: 110
match:
  vendor: ["Dell", "Dell Inc.", "Dell EMC"]
  firmware_patterns: ["iDRAC.*10", "^8\\.", "^9\\."]
capabilities:
  sections_supported: [system, hardware, bmc, cpu, memory, storage, network, firmware, power, thermal]
collect:
  strategy: standard_plus_oem
  standard_tasks: "redfish-gather/tasks/collect_standard.yml"
  oem_tasks: "redfish-gather/tasks/vendors/dell/collect_oem.yml"
normalize:
  standard_tasks: "redfish-gather/tasks/normalize_standard.yml"
  oem_tasks: "redfish-gather/tasks/vendors/dell/normalize_oem.yml"
```
