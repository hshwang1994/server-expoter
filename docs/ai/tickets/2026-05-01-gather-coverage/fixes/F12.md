# F12 — Cisco CIMC PowerSubsystem 검증

**우선**: P2 / **영역**: power / **출처**: R2 / **원칙**: Additive (검증만)

## 컨텍스트
Cisco CIMC 4.x 가 PowerSubsystem 응답하는지 미확인. 현재 R2 자료에서 명확하지 않음.

## 현재 동작
cycle 2026-05-01 `gather_power`:
- `/Power` 시도 → 200이면 사용
- 404면 PowerSubsystem fallback

→ Cisco CIMC가 어떤 응답하는지 lab 검증 필요.

## 검증 (코드 변경 없음)
1. Cisco CIMC 4.x 호스트 `curl -ku root:passwd https://<bmc>/redfish/v1/Chassis/1/PowerSubsystem` 시도
2. 200 응답이면 fallback 자동 동작 — 확인만
3. 404 응답이면 우리는 기존 Power 만 사용 — 영향 없음

## Cold-start
1. lab Cisco CIMC 호스트 접근
2. PowerSubsystem endpoint 확인
3. envelope `data.power.power_supplies` 채움 검증

## 회귀 risk
없음 (검증만, 변경 없음).
