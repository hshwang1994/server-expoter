## 컨텍스트
ESXi 가 InfiniBand 카드를 **Ethernet adapter 로 인식** (VPI mode default Ethernet, 또는 Mellanox OFED 별도 설치 필요).

→ ESXi 채널에서 IB 분리 수집은 vendor 정책상 **기술적 제약**. 사용자 명시 "기술적으로 안 되면 skip" 영역.
