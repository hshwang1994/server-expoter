# F29 — BIOS Attribute Registry 수집

**[STATUS]**: ⚠️ **호환성 영역 외 — 새 데이터** (사용자 의도와 별개). 별도 cycle.

**우선**: P3 / **영역**: bios (신규) / **출처**: R4 / **원칙**: Additive (단 새 데이터)

## 컨텍스트
현재 BiosVersion / BiosReleaseDate 만 수집. BIOS 설정값 (SecureBoot / Virtualization / TurboMode 등) 미수집.

## 변경 (Additive)
신규 섹션 'bios_attributes' 또는 system 섹션 안 sub-key

## 우선 P3 사유
운영 보안 정책 모니터링 요구 시 (SecureBoot 활성 여부 등)
