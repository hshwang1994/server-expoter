# F42 — Redfish v2 path 향후 호환성

**우선**: P3 (추적만) / **영역**: 횡단 / **출처**: R7 / **원칙**: 추적

## 컨텍스트
- Redfish URL 메이저 버전: `/redfish/v1/` (v1)
- 메이저 버전 변경 시 backward incompatible (DMTF 표준)
- 2026 현재 v2 미발표
- 우리 코드: `_p()` 함수가 `https://{bmc_ip}/redfish/v1/{path}` 하드코딩

## 우리 영향
`redfish_gather.py:90`:
```python
url = f'https://{bmc_ip}/redfish/v1/{path.lstrip("/")}'
```
v2 도입 시 BMC 가 `/redfish/v2/` 응답하는 펌웨어로 갱신되면 우리 코드 fail 가능.

## 변경 (추적만)
- DMTF Redfish release notes 추적 (분기별)
- 2025.4 / 2026.x 발표 시 v2 announcement 모니터링
- v2 도입 BMC 펌웨어 발견 시 fallback 패턴 적용 (Storage→SimpleStorage 모범)

## 추적 trigger
