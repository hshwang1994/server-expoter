# F05 — power EnvironmentMetrics fallback (PowerControl 보존)

**우선**: P1 / **영역**: power / **출처**: R1 DMTF / **원칙**: Additive

## 컨텍스트
DMTF 2020.4 PowerSubsystem 도입 시 PowerControl (system-level metric: PowerConsumedWatts / PowerCapacityWatts / PowerMetrics) 이 EnvironmentMetrics 로 분리됨.

cycle 2026-05-01 `_gather_power_subsystem` fallback이 PSU만 매핑 — PowerControl null.

## 현재 동작
`redfish_gather.py` `_gather_power_subsystem`:
```python
power_control = {
    'power_consumed_watts':  None,
    'power_capacity_watts':  pc_capacity,  # PSU 합산만
    'interval_in_min':       None,
    'min_consumed_watts':    None,
    'avg_consumed_watts':    None,
    'max_consumed_watts':    None,
} if psus else None
```

## 변경 (Additive)
```python
def _gather_power_subsystem(...):
    # ... 기존 PSU 매핑 ...

    # 신규: EnvironmentMetrics fallback
    em_path = _p(chassis_uri) + '/EnvironmentMetrics'
    st_em, em_data, _ = _get(...)
    if st_em == 200:
        power_control['power_consumed_watts'] = _safe(em_data, 'PowerWatts', 'Reading')
        power_control['min_consumed_watts'] = _safe(em_data, 'PowerWatts', 'ReadingRangeMin')
        power_control['max_consumed_watts'] = _safe(em_data, 'PowerWatts', 'ReadingRangeMax')
        # interval_in_min / avg_consumed_watts: EnvironmentMetrics 표준에 없음 — None 유지

    return {'power_supplies': psus, 'power_control': power_control}, errors
```

## 회귀
- 기존 Power 200 응답 BMC: 변경 없음 (PowerSubsystem fallback 진입 안 함)
- PowerSubsystem fallback 진입 + EnvironmentMetrics 200: PowerControl 정보 채워짐
- PowerSubsystem fallback + EnvironmentMetrics 404: PowerControl 그대로 PSU 합산만 (현재 동작)
