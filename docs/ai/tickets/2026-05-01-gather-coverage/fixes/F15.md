# F15 — Supermicro X9 adapter 정확성

**우선**: P3 / **영역**: system / **출처**: R2 / **원칙**: Additive (capability 정확성)

## 컨텍스트
Supermicro X9 (ARM11 BMC) 는 Redfish 사실상 미지원. 현재 adapter `supermicro_x9.yml` 가 어떤 capabilities 명시했는지 검증 필요.

## 현재 동작
확인 필요 (`supermicro_x9.yml` priority / sections_supported).

## 검증 후 변경 (해당 시)
- sections_supported 비우거나 generic 수준으로 strict
- 호스트가 X9 일 때 status='failed' 정상 분류

## Cold-start
1. `adapters/redfish/supermicro_x9.yml` 검토
2. Supermicro 공식 X9 Redfish 지원 매트릭스 확인
3. capability 정정 (해당 시)
4. baseline 갱신 (supermicro_x9_baseline.json — 비우거나 minimal)

## 회귀 risk
adapter capability 변경 시 일부 vendor 매칭 변동 가능. priority 차등 확인 필수 (X9 는 priority 0~10 generic fallback 수준).
