# F09 — HPE Gen10/Gen10+/Gen11 BIOS Oem path 변경

**우선**: P3 / **영역**: system / **출처**: R2 / **원칙**: Additive (현재 영향 없음)

## 컨텍스트
HPE iLO 5 BIOS endpoint 위치 변경:
- Gen10: `/Systems/1/Bios/Service`
- Gen10+/Gen11: `/Systems/1/Bios/Oem/Hpe/Service`
- Gen10+/Gen11: `/Systems/1/Bios/Oem/Hpe/BaseConfigs`

## 현재 동작
우리 코드는 `/Systems/{id}/Bios` endpoint 직접 호출 안 함. `data.system.Oem.Hpe` namespace 만 read (System schema 안 OEM)

→ **현재 영향 없음**

## 변경
없음 (등재만).

## 향후 검토 (R4)
BIOS attribute registry 수집 도입 시 — Gen10 vs Gen10+ path 분기 필요
- vendor 분기 (rule 12 R1 — OEM Allowed)
- 우선: P3 (사용자 요구 시)
